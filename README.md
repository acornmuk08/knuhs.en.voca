# 심화영어작문 VOCA 입니다.

## 개요

**이 프로그램은 html과 javascript로 동작합니다.**
영어 단어를 포집하고 화면 기본 ui를 만드는 과정에서 _Gemini_ 의 도움이 있었으며
스프레드시트 크롤링 등의 핵심 과정은 순수 자작입니다.
모든 문제는 5지선다 입니다.

스프레드시트 링크는 https://docs.google.com/spreadsheets/d/1UDeX4SexApThjVkLnfjmKc9DWC2yTtAj85ldyo63L_I/ 이며, 혹시나 추가되지 못한 단어가 있을 경우

| 영어 | 한글뜻 | 영영풀이 |
|---|---|---|
| 영어 | 한글뜻 | 영영풀이 |

형식으로 추가하시면 됩니다.

## 기능설명
현재 이 프로그램은 다음과 같은 문제 출제 방식이 있습니다.
* 한국어 뜻 보고 영어 맞추기
* 영어 스펠링 보고 한국어 맞추기
* 영어 보고 영영풀이 맞추기
* 영영풀이 보고 영어 맞추기
* 단어 리스트 보기 _06-25 추가_

## 기타

* 중간 정지가 가능합니다. **다만 중간 정지 이후 초기화 하여 다시 문제를 풀면 문제 순서가 바뀝니다!**
* 다크모드를 지원합니다.
